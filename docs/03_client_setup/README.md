# 3. Angular View Setup

## New angular client
```ps
client $ ng new client --strict false
? Would you like to add Angular routing? Yes
? Which stylesheet format would you like to use? CSS
```

## Run the client
```ps
client $ ng serve
```

## Install Angular bootstrap
```ps
client $ ng add ngx-bootstrap
```
